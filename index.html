Getting Started with Dot1Xer Supreme

Welcome to Dot1Xer Supreme, the comprehensive tool for 802.1X configuration management. This guide will help you get started using the application.

Introduction to 802.1X

IEEE 802.1X is a standard for port-based network access control that provides an authentication mechanism for devices wishing to connect to a LAN or WLAN.

The main components in an 802.1X deployment are:

  • Supplicant: The client device requesting access to the network
  • Authenticator: The network device (switch or access point) that enforces authentication
  • Authentication Server: Typically a RADIUS server that validates credentials

Using the Configuration Tab

The Configuration tab is where you'll generate 802.1X configurations for your network devices:

  1. Select your network device vendor (Cisco, Aruba, etc.)
  2. Choose the specific platform or OS version
  3. Enter your RADIUS server details
  4. Configure VLANs for authenticated, guest, and voice devices
  5. Specify the interfaces where 802.1X will be enabled
  6. Adjust advanced options as needed
  7. Click "Generate Configuration" to create your configuration

Next Steps

After generating your basic configuration, explore these features:

  • Use the Templates tab to access pre-built configurations
  • Try the Discovery tab to identify devices on your network
  • Use the AI Assist feature to analyze and improve your configurations
